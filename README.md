
====================================================================


We implemented our MatGSW bootstrapping scheme to testing the correctness.



Instructions
------------
Install g++,cmake,make. Compilation has been tested using g++ on Linux.

To compile:

	mkdir build
    cd build
    cmake ..


To run main program:

	make
    ./MatGSW
